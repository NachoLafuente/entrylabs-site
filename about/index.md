## Why this exists

The old Entry Labs was a self-serve SaaS. It worked technically. It didn't work commercially. CEOs approve "AI consulting / agent deploy" line items. Procurement blocks new SaaS subscriptions.

So we rebuilt it as a productized service. Same extraction kernel from v0.9 (which actually shipped to production), wrapped in a "we build, we deploy, we operate" model. Fewer dashboards. More working software.

## Where we sit in your stack

We're not a CRM. We're not a SaaS dashboard. We're not another Slack app you have to maintain. We're the team that takes your Attio plus your meeting-note tool plus your enrichment APIs and makes them act like a single agent on your behalf.

The agent runs on infrastructure we deploy and own for you. You own the data, the API keys, the Attio fields. We own the "does it work" problem.
